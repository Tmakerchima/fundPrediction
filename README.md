# FundLens MVP

一个覆盖中国场外公募基金、场内 A 股和美股的趋势研究网页。顶部可以切换三个独立市场页面；基金页使用 5–60 个交易日预测，股票页使用 10 个交易日研究窗口。

## 快速启动

需要 Node.js 20+，不需要安装第三方依赖。

```bash
npm start
```

打开 `http://127.0.0.1:4173`。开发模式使用 `npm run dev`，测试使用 `npm test`。

### 配置 Qwen 证据审查

密钥只从服务端环境变量读取，不能放进前端或提交到仓库。PowerShell 启动示例：

```powershell
$env:DASHSCOPE_API_KEY="替换为已轮换的新密钥"
$env:QWEN_MODEL="qwen-plus"
npm start
```

股票详情和基金详情中可以粘贴财报、公告、季报或新闻原文。Qwen 只返回事实、风险、催化因素以及它与量化信号是否冲突；原始预测和回测值保持只读。当前 MVP 不会自行联网寻找公告，避免把搜索摘要或不明来源误当成正式披露。

## 模型说明

- 数据：东方财富公开基金历史净值，后端缓存 10 分钟。
- 原始信号：分别计算最近 20、60、252 个交易日（约 1、3、12 个月）的平均对数趋势并等权合成。该组公开期限参考 AQR 的多期限时间序列动量研究，不使用或声称获得任何量化公司的私有参数。
- 样本校准：使用不重叠的历史预测窗口，以过往“原始预测 × 实际收益”的过原点回归斜率校准当前预测；斜率限制在 `[0, 1]`，因此只能把不稳定预测向零收缩，不能放大，也不会在趋势失效时偷偷反转成均值回归策略。历史回测中的每个斜率只使用该时点以前的数据。
- 区间：用最近 60 个交易日对数收益波动生成 80% 与 95% 经验区间。
- 验证：最近样本的走步式样本外验证，展示 MAPE、相对随机游走的样本外 R²，以及模型在历史上发出看涨信号时的条件胜率与 Wilson 95% 区间。看涨概率使用 `(胜利次数 + 1) / (看涨次数 + 2)` 做简单收缩，避免少量样本直接显示 0% 或 100%。
- 基准：展示相对“未来净值等于当前净值”随机游走基准的样本外 R²；负数表示模型不如朴素基准。
- 评分：不再使用主观加减分表；评分 = `NormalCDF(预测对数收益 / 预测期波动) × 100`，解释为模型分布假设下的盈利概率，而不是收益保证。
- 费用：主页读取东方财富公开费率页，按计划持有天数匹配赎回费，并使用页面展示的申购优惠费率计算费用后预测；下单前仍需核对实际销售平台。
- 组合：低波动组合先比较下行波动与回撤，均衡组合比较费用后收益/风险，趋势组合比较费用后预测收益；权重使用逆波动率并计入历史相关性。
- 调仓：预测信号随净值更新，但交易组合按自然周锁定。新标的需越过持仓缓冲，权重每次只向新目标移动 50%；不足三只通过硬门槛时保留现金。

## 近两周研究组合

首页组合先从股票、混合、指数、债券和 QDII 五类基金候选中预筛选，排除名称中明确带养老、封闭或长期持有安排的产品，再运行 10 个交易日预测、真实费率扣减、样本外验证和风险过滤。证据分为三级：A级要求费用后预测为正、样本外 R² 为正、至少 20 个历史看涨样本且“看涨后实际上涨”胜率的 95% Wilson 区间下限不低于 50%；B级保留前述费用、R²和样本要求，但看涨条件胜率改为不低于55%；C级是更宽松且明确标注高风险的观察候选。A级不足时可以用B/C级补充，但系统会自动降低风险预算并保留更多现金，页面必须说明仍可能亏损。

公开方法来源：Jane Street 的公开材料只覆盖概率、期望值和做市基础，并未披露生产交易参数；AQR 公开了 1/3/12 月时间序列动量期限及跨市场长期实证；Man AHL 公开说明了波动率缩放与快速趋势在成本上的权衡；Two Sigma Factor Lens 和 Microsoft Qlib 强调把预测与因子暴露、滚动训练、IC 和组合回测分开验证。本项目只实现这些可审计原则，不冒充复刻机构模型。

已有持仓可以保存在浏览器本地，系统会按照买入日期匹配当前赎回费率并独立给出“持有、不加仓或复核退出”；掉出主页前三不再被当作卖出信号。周度状态默认写入 `.runtime/recommendations.json`。Railway 等无状态部署应把持久卷挂载到该目录，或设置：

```powershell
$env:RECOMMENDATION_STATE_PATH=".runtime/recommendations.json"
```

这个过程仍存在幸存者偏差、动量预筛选偏差和多重检验问题。当前单基金回测没有覆盖“在每个历史时点重新对全市场选出前五名”的完整策略，因此榜单属于研究关注池，不是经过验证的可交易策略。

## 重要限制

基金净值通常仅在交易日收盘后更新，无法像股票一样实时成交。场外基金采用未知价原则，信号使用的最后净值不等于下一笔申购的确认净值。费率页也可能与实际渠道优惠不同。预测仍不包含底层持仓实时变化、基金经理变更、宏观事件或个人完整财务情况；页面输出不构成投资建议，不保证未来收益。

## A 股与美股页面

- A 股：`/a-shares.html`，腾讯前复权日线，15 支高流动性代表股研究池。
- 美股：`/us-stocks.html`，Nasdaq 历史日线，15 支高流动性大型股研究池。
- 策略：SMA50/200 趋势过滤、风险调整概率预测、行业去重、交易成本和非重叠样本外验证。
- 机构参考：只采用 State Street、JPMorgan 和 Jane Street 对外公开的多因子、组合风险、流动性与执行原则；其真实生产模型均为专有资产，本项目没有也不声称复制。
- GitHub 工程参考：Qlib 的研究工作流，以及 vectorbt / Backtesting.py 的可复现回测思想。GitHub 热度不能证明策略具有未来超额收益。

股票回测尚未完整模拟滑点、税费、涨跌停、停牌、汇率和实际成交。推荐卡片是研究候选而不是买卖指令；样本外 R² 小于或等于 0 时，模型未战胜随机游走基准，应降低或放弃该信号。
